# Tái hiện thu gọn và xây dựng quy trình forensic phát hiện Linux desktop espionage bằng memory forensics

### Tài liệu tham khảo 
[1] Paper: [Uncovering linux desktop espionage](https://doi.org/10.1016/j.fsidi.2025.301921)
[2] Repository: [uncovering-linux-desktop-espionage](https://github.com/FHMS-ITS/uncovering-linux-desktop-espionage)

### Yêu cầu môn học
Xem file requirements/requirements.md

### Quy trình thực hiện
Gồm 3 giai đoạn (3 stages):
- Giai đoạn 01: Reproduce bằng memory dumps có sẵn
- Giai đoạn 02: Tái hiện thực nghiệm qua các case study tự dựng
- Giai đoạn 03: Cải tiến đề xuất và thực nghiệm kiểm chứng

Cả 3 giai đoạn vừa thực hiện vừa viết báo cáo 

### Cấu trúc báo cáo
* Chương 1: Giới thiệu
* Chương 2: Nghiên cứu liên quan
* Chương 3: Cơ sở phương pháp
* Chương 4: Tái hiện dựa trên các case study
* Chương 5: Đánh giá phương pháp gốc và so sánh với baseline
* Chương 6: Đề xuất cải tiến và kiểm chứng thực nghiệm
* Chương 7: Kết luận
